## How to Remove Default List Style from ul in CSS ?

I refer the code from the article shared by [GeeksforGeeks](https://www.geeksforgeeks.org/how-to-remove-default-list-style-from-unordered-list-in-css/).

```css
ul {
    list-style: none;
    padding: 0;
    margin: 0;
}

li {
    margin: 0;
}
```
